Project Title:
Advanced 2D/3D Electrical & Electronics Circuit Builder, Analyzer, and Real-Time Simulator

Project Overview

Build a professional-grade web-based Electrical and Electronics Circuit Design, Visualization, Analysis, and Simulation Platform using React, TypeScript, Three.js, React Three Fiber, and modern web technologies.

The application should allow users to build circuits by drag-and-drop, simulate real-world electrical behavior, visualize current flow in real time, identify design mistakes, measure outputs using virtual instruments, and switch between schematic (2D) and realistic (3D) views.

The goal is to create a browser-based alternative inspired by:

- LTSpice
- Multisim
- Proteus
- Tinkercad Circuits
- Falstad Circuit Simulator
- EasyEDA
- KiCad
- Fritzing
- CircuitLab

while providing a highly interactive infinite-canvas 3D experience.
